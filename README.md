# Challenge by Coodesh

## Objetivo do Projeto

O objetivo central deste projeto é permitir a realização de pedidos de forma simples e eficiente. 
A aplicação foi desenvolvida para demonstrar habilidades em .NET, proporcionando uma interface gráfica intuitiva para cadastrar, listar e gerenciar pedidos, 
além de cadastrar pessoa e produtos, e por fim persistir os dados em um arquivo JSON.

## Tecnologias Utilizadas

- .NET 10
- C#
- WPF (Windows Presentation Foundation)
- MVVM (Model-View-ViewModel)
- Serialização JSON

## Como Executar o Projeto

1. **Pré-requisitos:**
   - Visual Studio 2022 ou superior
   - SDK do .NET 10 instalado

2. **Clonar o repositório:**
   ```bash
   git clone https://github.com/danillo-fullstack/challange-by-coodesh.git
   ```

3. **Abrir o projeto:**
   - Abra o arquivo `challange-by-coodesh.sln` no Visual Studio.

4. **Restaurar dependências:**
   - O Visual Studio irá restaurar automaticamente as dependências ao abrir o projeto.

5. **Executar a aplicação:**
   - Pressione `F5` ou clique em "Iniciar" para rodar o projeto.

6. **Utilizar o sistema:**
   - Utilize o formulário para realizar cadastrar pessoa, cadastrar produto e realizar pedidos, que serão exibidos em um grid e salvos em um arquivo JSON na pasta `Data`.
